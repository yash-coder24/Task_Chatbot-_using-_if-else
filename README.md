# Chatbot-_using-_if-else
Developed a console-based chatbot using Python with if-else logic and input loops. Implemented rule-based responses, date/time handling, and conversational flow to simulate basic natural language interaction

# Objective
Create a console chatbot that:-
Takes user input,
Responds using if-elif-else rules,
Runs in a loop.

# Tools
Python ,
Terminal / VS Code

# Example Conversation
ChatBot: Hello! I am your chatbot. Type 'bye' to exit.
You: hi
ChatBot: Hello there! How can I help you?
You: what is your name
ChatBot: I am a simple Python chatbot.
You: time
ChatBot: Current time is 14:32:10.
You: bye
ChatBot: Goodbye! Have a nice day .
